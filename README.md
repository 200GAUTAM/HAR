# HAR
## Human Activity Recognition

A learning algorithm to predict human activity recognition using the sensory data from a smart phone. There are 6 activities to be predicted which include walking, walking upstairs, walking downstairs, sitting, standing and laying.

"README.txt" under datasets has the detailed information regarding the data and its structure. The original data is here  - http://archive.ics.uci.edu/ml/datasets/Smartphone-Based+Recognition+of+Human+Activities+and+Postural+Transitions

The learning model is devised by a three layer neural network. Sigmoid function is used as the activation.
The model is trained for about 100 iterations with regularization parameter set to 1.

Training set accuracy ~ 97 percent

Test set accuracy ~ 94 percent
